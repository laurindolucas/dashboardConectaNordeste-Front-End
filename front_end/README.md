## Dashboard de Visualização de Dados do Censo Escolar 2024

Este repositório contém o código-fonte e a documentação do Dashboard de Visualização de Dados do Censo Escolar 2024.

O projeto foi desenvolvido como um trabalho prático para a disciplina de Engenharia de Software e tem como principal finalidade oferecer uma ferramenta visual e interativa para a análise dos dados provenientes de outra disciplina.
### Visão Geral do Projeto

O dashboard tem o papel de consumir os dados do Banco de Dados gerenciado na disciplina do Professor Gabriel.

- Fonte de Dados: O banco de dados é baseado em um dataset real do Censo Escolar Brasileiro de 2024.

- Finalidade: O principal objetivo é visualizar e explorar de forma mais eficiente e compreensível as informações contidas no banco, utilizando gráficos, tabelas e filtros, transformando dados brutos em insights acionáveis.

## Tecnologias Utilizadas no Front-end

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
